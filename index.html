About me

Alright, I'll keep it short. I'm a programmer who dreams of becoming a video game developer. I'm not the best at this, but I love what I do, and I hope to make new players feel what I felt when I started my first game of Pokémon Gold on my pink Game Boy Advance.

I am currently studying Video Game Design and Development at Rey Juan Carlos University in Madrid. I am proficient in using the Unity game engine, as well as programming languages like C, C++, C#, Java, and JavaScript.

Game Jams

The Inner Doors
2nd place in the Game Scholars 3rd Edition Game Jam

A 3D mystery first-person game where you have to discover what is happening and where your decisions matter. You can get 3 different endings.

In this game, my job was to create all the visual aspects of the game (from the interface, to all the 3D assets).

Try the game here: The Inner Doors

Regrets & Complaints
2nd place in the Game Scholars 4th Edition Game Jam

A 2D graphic adventure where you meet funny and sinister characters.

In this game, my work focused on setting up the dialogue system as well as creating the shaders for the visual style.

This game is only available in Spanish.

University projects

Space Combat Rush 3D

It's a multiplayer browser game made in Unity, inspired by another project developed during my studies, Space Combat Rush.

It's a one-on-one online fighting game featuring spaceships, where players can customize their ships both in terms of equipment and appearance.

For this game, I was primarily responsible for managing the connections, the client-server structure, ensuring the proper functioning of users and their data, as well as the game logic.

The game is still in development.

Try the game here: Space Combat Rush 3D

The Inner Doors VR

It's a VR demo featuring the first two days of the game Inner Doors, which you can find above.

It includes the first two days adapted to VR, with adjustments for movement, interaction, and mini-games.

My work here includes everything related to making the game functional in virtual reality, from adapting controls, movement, accessibility, and interaction to redesigning mini-games to ensure they are interactive in VR.

Try the demo here: The Inner Doors VR

EggQuest

EggQuest is a platformer and action game for Gameboy.

This project is currently purely conceptual. It is part of a course in the art and design field, and since it's not my area of expertise, I chose the aesthetic of the Game Boy.

S.P.A.D.E.

A 3D platform game following the idea of hard games that punish you for any and every mistake. You have to help Foggy reach the top but, jumping over the platforms will be a challenge because, if you fall, you have to start it all over again.

This game is still in progress. My work has been focused on creating the collectibles system as well as some shaders to achieve the visual style.

Try the demo here: S.P.A.D.E.

Space Combat Rush

A 2D arcade game in pixel art to reminiscence the old times. It was created as our first multiplayer game. Here, the objective is to kill your adversary and survive in a space war.

This time, my work involved programming the REST API, the messaging system, and the game structure so that my teammates could complete it.

Try the game here: Space Combat Rush
